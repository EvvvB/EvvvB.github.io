Temperature Chart

Tech-Stack : Javascript, HTML, CSS, Node.js, Express, Mongodb, Python, charts.js

Description:

This webapp functions as a tool to visualize the temperature in my room for the past 72 hours. Using a raspberry pi and a ds18b20 sensor, hourly data is sent directly from the raspberry pi to my server through a python script which is run through a cron job. The node.js server takes this data storing it into a mongodb server, though the server is not directly accessed at the moment. In order to preserve the database requests there is a cache of the latest values on the node server.
