Temperature Chart

Historical personal project; the hosted demo is no longer running.

JavaScript, HTML, CSS, Node.js, Express, MongoDB, Python, Chart.js

This web application functions as a personal tool to visualize the temperature in my room for the past 72 hours. Using a raspberry pi and a ds18b20 sensor, hourly data is sent directly from the raspberry pi to my server through a python script which is run using a cron job. The node.js server takes this data storing it into a MongoDB server. Though, the database is directly accessed with each page load at the moment. Instead, in order to preserve the database requests and speed up loading time, there is a cache of the latest values on the server instead.
